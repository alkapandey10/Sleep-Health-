Sleep Health Lifestyle Prediction using Machine Learning:- 

Project Overview:-
This project focuses on analyzing and predicting sleep health and lifestyle patterns using Machine Learning techniques. The model uses lifestyle and health-related factors such as stress level, physical activity, BMI, heart rate, and daily habits to predict sleep-related outcomes.

The objective of this project is to understand how different lifestyle factors affect sleep health and to build a predictive model with strong accuracy.

Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Data Visualization
Machine Learning Model Building
Model Evaluation using Performance Metrics

Model Performance:-
Metric  Score
R2 Score  0.8512
MAE       0.2498
RMSE     0.3147

The model achieved strong predictive performance and explains around 85% variance in sleep health outcomes.

Results:-
The project successfully predicts sleep health outcomes using lifestyle and health-related data. Feature analysis and visualizations helped identify important factors affecting sleep quality.
